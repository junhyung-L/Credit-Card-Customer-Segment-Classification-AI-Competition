# 💳 Credit Card Customer Segment Classification
## Data-Driven Customer Profiling & Behavioral Analysis

### 📌 Project Vision
Segmenting financial customers based on refined behavioral datasets to optimize marketing budget and improve user experience. This project uses multi-class classification to identify high-value vs. low-engagement users.

### 🛠️ Key Approaches
- **Feature Engineering:** Temporal spending patterns and categorical categorization.
- **Ensemble Modeling:** High-AUC classification using Random Forest and LightGBM.
